結合RetinaFace與UP-LBP自動辨識Apex幀的微表情的研究

作者:丁全宏  指導教授:蘇溢芳

論文連結

簡介

  微表情是一種持續時間短的一種反射動作，難以去做偽裝。在過去的心理學研究中，也證實微表情因為其特性可以作為一種有效辨識情緒的方法。在微表情的持續時間之中，頂點幀(Apex)其中表情最豐富變化最大的那一幀，也可以藉由頂點幀(Apex)來去提升情緒辨識的效率。
  然而多數資料集皆沒有提供頂點幀，所以我們的目的是要在一段含有一個微表情影片，並從其中的特徵找出頂點幀，我們的方法主要分成三個部分，臉部擷取，減少資料受到背景與肢體等的影響；找出頂點幀範圍，我們根據每兩幀做比較，去避免在過去的研究中，使用易受外在因素影響的參考幀；找出頂點幀，透過觀察影片中微表情的變化，通過眾數從前一步結果的兩個幀找出我們的頂點幀。
  我們的研究結果顯示，相較於過去的研究，我們的研究更能在一段含有一個微表情影片正確找出頂點幀，且所花費時間也比他們少。

關鍵詞 : 微表情辨識、Uniform Pattern LBP 、N X N 不相交、特徵差距、頂點幀定位

過去的研究


  (1)透過資料集標籤或是Apex Spotting找到的Apex Frame進行情緒分析

  資料集標籤提供的Apex標籤多為人工,容易受標註者主觀意識影響，且多數沒有提供Apex標籤，如DAiSEE等

  特徵提取:透過每一幀與第一幀做比較，找出差距最大的值，那個幀就是Apex幀

  方法直接把第一幀當作沒有微表情幀(natural frame)，在封閉的實驗確實可以，但現實生活中影片不一定第一幀就是natural frame

  特徵提取+資料集標籤:透過資料集提供的onset幀到offset幀之間，做特徵提取把所有特徵質做平均

  特徵提取+資料集標籤:每一幀與平均值做比較，找出差距最大的值，那個幀就是Apex幀

  微表情定義

  造成的問題:如圖所示。第一幀（起始幀）和最後一幀（結束幀）與平均值的差距都是141，因此都會被判定為頂點幀，而這樣的結果違背了微表情的基本原理。由於微表情是一個動態變化的過程，頂點幀應該出現在表情變化的過程中，而不是在開始(onset)或結束時刻(offset)。且根據微表情的定義，起始幀代表表情開始變化的時刻，結束幀代表表情恢復平靜的時刻

本研究主要面臨的挑戰

  1.臉部擷取的準確性問題

  2.高幀率影片中Apex frame的識別效率問題

  3.自動辨識關鍵幀的問題

Problem Definition

  find:微表情Apex幀

實驗


Dataset

  多數資料集為提供Apex、Offset與Onset幀的標籤，所以選擇以下的資料集來測試我們的方法，並透過提取特定範圍的幀來模擬其他未提供微表情幀標籤的資料集。每個實驗為了準確性，會將30次的實驗結果作平均。

使用的資料集


資料集是以圖片的方式儲存，每張圖片代表一個幀，例如一個1秒100幀的影片會有6000張圖片


一般資料集沒有提供onset跟offset幀的兩個標籤，如DAiSEE，所以我們根據這兩個標籤往前跟往後提取固定幀數來模擬其他資料集的狀況。且由於兩個資料集的幀數不同，我們採用不同的幀數提取各自的圖片，但實際時間都大約為1.2秒

SAMM的資料集:透過資料集提供的標籤，往前往後各提取250幀，作為實驗的影片


SMIC_HS的資料集:透過資料集提供的標籤，往前往後各提取120幀，作為實驗的影片


SAMM的的實驗結果: 根據上圖，我們的方法優於其他方法，但是Cube-LBP的實驗結果與我們很接近，且時間更少

SMIC的實驗結果: 根據上圖，我們的方法依舊優於其他方法，而Cube-LBP的方法的正確率則明顯的下降，兩個資料集的差別主要在幀數，而Cube-LBP根據前後幀提取時空特徵，幀數減少導致正確率下降
